ZIGBEE BASED SECURITY SYSTEM

The purpose of this project is to implement an interesting and inexpensive home security alarm. This gadget helps us to protect our house from thieves. This is a simple motion detector alarm that detects when someone enters the area. When an intruder is detected, itactivates a siren. The receiver circuit consist of a Zigbee module, an LCD, and a GSM module for notifications. The communication between the two circuits is done by Zigbee Module. Whenever there is a motion in any of the locations or an alert of window getting open than a notification will sent to the registered user
